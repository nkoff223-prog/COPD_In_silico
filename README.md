# COPD_In_silico
COPD associated proteins were docked against 5 suitable phytocompounds of Durva using SwissADME, Excel, PyRx and BioVia Discovery Studio. 
5 Ligands were docked against 3 COPD associated proteins.
MMP9,IL8 and TNFalpha- 3 COPD associated proteins.
5 Ligands- Apigenin,
Binding affinity of -10.3kCal/mol observed with Apigenin and MMP9.
